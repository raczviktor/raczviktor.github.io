Technology Wiki

Logic Gate

An idealized or physicalphysical device that performs a basic logical operation on one or more binary inputs and produces a binary output.

ASCII

As English name is American Standard Code for Information Interchange. Computers only understand numbers, so it describes the numbered values ​​needed to interpret the characters 'a' or '@'.

RAM

A random access read / write storage device. Stores the programs to be executed by the CPU and the data to be processed. The speed at which a particular piece of memory is accessed is independent of its location.

Algorithm

A series of instructions used in computing that can be read and interpreted by man to communicate directly or indirectly (for example, after translation into machine code) to a computer how to perform a particular task.

DNS

A hierarchical, highly decentralized naming system for computers, services, and any resource connected to the Internet or a private network.

HTTP

An application protocol for distributed, collaborative and hypermedia information systems. The basis of data communication on the World Wide Web. This is the basic protocol for exchanging and moving hypertext.
