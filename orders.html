Cmd parancsok


Cmd parancsokmkdir xyz: xyz mappa létrehozása
cd xyz: belépés az xyz mappába
cd..: visszalépés
dir: mappa tartalmának kilistázása
echo szöveg >> fájlnév.txt : txt fájl létrehozása és szöveg beleírása

Git parancsok a cmd-ben (a repo mappában)


git clone URL: teljes repo klónozása
git status: git állapota
git add fájlnév vagy .(összes): fájl hozzáadása a stage státuszhoz
git commit –m „megjegyzés”: módosítások mentése
git fetch: serveren lévő fájlok összehasonlítása a gépen lévőkkel
git pull: serveren lévő fájlok lementése a gépre
git push: a gépen lévő fájlok feltöltése a szerverre
