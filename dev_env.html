Linkek a fejlesztői környezet kialakításához.

  • Google Chrome (böngésző)
  • Visual Studio Code (szerkesztő):
    szükséges bővítmény (extensions ctrl+shift+x): live server
  • Node.js (javascript):
    ellenőrzés: cmd-> node –v (verziószám)
  • Git:
    ellenőrzés: cmd-> git (használati utasítás)
  • Github:
    • új repository létrehozása
    • új repository klónozása: code -> Url másolása -> cmd git clone „URL cím”-> cd „repo neve”-> code . (megnyitás Visual Studioval)
    • repository fork-olása: kiválasztott repo-nál fork-> létrehozott saját repo code -> Url másolása -> cmd git clone „URL cím”-> cd „repo neve”-> code . (megnyitás Visual Studioval)
