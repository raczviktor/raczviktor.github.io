CSS

Fajtái

Inline stílus: elemenként megadva.

  • Az elemben deklarálva.
    pl: style="color: színszín;"

Internal stílus: kiszervezve a head részbe.

  • A head részben deklarálva.
    pl: style h5 {color: színszín;} /style.

External stílus: kiszervezve külön oldalra.

  • Külön css kiterjesztésű oldalon deklarálva. A kívánt oldalon, a head részben linkelve a link rel="stylesheet" href="CSS FÁJL HELYE"

    pl: style p {color: színszín; font-size: SZÁMpx;}

Felépítés (szintaxis)

Direktíva

  • Selctor (mire vonatkozik)
  • Deklaráció (mit tegyen vele)

Betűtípus

  • Beállítása: font-family: BETŰTÍPUSNEVE
  • Külső betűtípus használata
    1. GoogleFonts
    2. Ékezetes karakterek teszteléséhez: Árvíztűrő tükörfúrógép.
    3. Letöltés, vagy Link elhelyezése a html-ben, vagy css-ben.

Saját betűtípus használata

  • @font-face {font-family: "KÍVÁNTNÉV"; src: url("ELÉRÉSIÚT");}

Szövegformázás

  • betűtípus/font-family
  • betűszín/color
  • betűméret/font-size
  • szöveg igazítás/text-align
  • szöveg átalakítás/text-transform
  • sormagasság/line-height
  • betűvastagság/font-weight
  • elemek megjelölése
    • egyedi azonosító/id: id="KÍVÁNTNÉV", css-ben #MEGADOTTNÉV {DEKLARÁCIÓ}
    • osztályokba sorolás/class: class="KÍVÁNTNÉV", css-ben .MEGADOTTNÉV {DEKLARÁCIÓ}
  • szöveg tagolás
    • szövegrész osztályba sorolás/span class
    • terület osztályba sorolás/div class
