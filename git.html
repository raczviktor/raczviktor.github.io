Git

Fájlok állapota a git-ben (add/commit/remove)

  • Untracked: nem figyelt
  • Unmodified: nem módosított
  • Modified: módosított
  • Staged: mentésre előkészített

Személyes oldal létrehozása Github-bal.

  1. repo létrehozása (A név kötött: felhasználónév.github.io)
  2. repo klónozása (Lásd: Fejlesztői környezet)
  3. fájlok bemásolása a létrehozott mappába
  4. cmd git add .
  5. cmd git commit -m MEGJEGYZÉS
  6. cmd git push
