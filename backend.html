Backend

Telepítés egy helyre

  1. Telepítsd a Node.js-t
  2. Cmd-vel lépj be a kívánt mappába
    • npm i json-server
  3. A telepítés helyén hozz létre egy új js fájlt.
    • const json-server = require("json-server");
  4. A mappában használható a json-server.

Telepítés Globálisan

  1. Telepítsd a Node.js-t
  2. Cmd
    • npm i json-server -g
    • Globális telepítés esetén a C:\Users\Aktuális felhasználó\AppData\Roaming\npm\node_modules\json-server helyen jön létre.
  3. Az egész gépen használható az json-server.

Package telepítése

  1. Cmd-vel lépj be a kívánt mappába.
    • npm init -y

Package törlése

  1. Cmd-vel lépj be a kívánt mappába.
    • npm remove json-server
  2. A json-server package eltávolítva.

Használat (jason-server)

  • Tetszőleges nevű és tartalmú .json fájl létrehozása egy mappában.
  • Elindítani a cmd-ből, bárhonnan: json-server --watch FORRÁS

JSON

  • közös nyelv
  • ellenörzéshez: https://jsonformatter.curiousconcept.com/
  • a kulcsokat is "jelbe kell tenni
  • a boolean-t és a numbert nem kell idézőjelbe tenni
  • json tartalom js-ben kezelése JSON.parse
    • példa
      • let json = `{ "users": [ {"id": 1, "name": "Big Tom", "email": "bigtom@gmail.com" }, {"id": 2, "name": "Little Jimmy", "email": "liljim@gmail.com" }, {"id": 3, "name": "Mid John", "email": "midjohn@gmail.com" } ], "comments": [ {"id": 1, "body": "some comment", "userId": 1} ] }`;
      • JSON.parse(json);
  • js tartalom json-ben kezelése JSON.stringify
    • példa
      • let json = [ {surname: "Berger", firstname: "Whitney", age: 22}, {surname: "Kovács", firstname: "Béla", age: 32}, {surname: "Rostás", firstname: "Márió", age: 12}, {surname: "Piros", firstname: "Gizella", age: 43}, {surname: "Zöld", firstname: "Gábor", age: 42}, {surname: "Maros", firstname: "Péter", age: 41}, {surname: "Gelencsér", firstname: "Lajos", age: 31}, ];
      • JSON.stringify(json);
